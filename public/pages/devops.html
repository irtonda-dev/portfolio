What's happening here

This project runs as two Docker images side by side. The old version is the original site — plain white, black text, no styling. The current version is what you're looking at right now.

Hitting Update below will pull the latest Docker image from the registry and restart the container automatically. Hitting Rollback swaps back to the previous image instantly. Either way — refresh your browser after a few seconds.
